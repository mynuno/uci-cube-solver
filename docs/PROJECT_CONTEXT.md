# UCI Cube Solver 프로젝트 맥락

## 목적
UC Irvine 그림 큐브의 6면 사진을 입력받아 원근 보정, 조각 분류,
유효성 검사, 3×3 풀이, 단계별 회전 안내를 제공하는 모바일 중심 웹앱.

## 기술 스택
- React
- TypeScript
- Vite
- cubejs
- Vitest
- CSS

## 목표 사용자 흐름
1. 6면 사진 등록
2. 모서리 지정 및 3×3 분할
3. 54개 스티커 목표 면·위치·회전 지정
4. 완성 이미지 및 큐비 유효성 검사
5. facelet 문자열 생성
6. 풀이 계산
7. 단계별 회전 안내
8. 완료 확인

## 면 기준
- U: 위
- D: 아래
- F: 앞
- B: 뒤
- L: 왼쪽
- R: 오른쪽

회전 방향은 해당 면을 정면에서 바라본 기준.

## 현재 제품 단계
기능 프로토타입 완료.
완성형 제품을 위한 구조 리팩터링, 모바일 UX, 저장·복구,
센터 방향 처리, 오류 복구, 배포가 남아 있음.
